How It Works
Awaaz-Seal utilizes Digital Signal Processing (DSP) to secure audio:
1. Encoding: We inject a 19kHz high-frequency sine wave into the audio data. This frequency is typically inaudible to adults but easily detected by algorithms.
2. Decoding: The system uses a Fast Fourier Transform (FFT) to analyze the frequency spectrum and verify the presence of the 19kHz seal.

Technical Logic
The protected signal follows a mathematical addition:
Protected Signal = Original Signal + Inaudible Seal (19kHz Sine Wave)

Frequency:19,000 Hz
 Low-decibel injection to ensure it remains silent to the human ear.
